About and how to use:

  • Colors are generated at random using vanilla JavaScript.
  • Select the refresh icon to view a new set of colors or sort the current grid by hue.
  • To add a color to your favorites, select its plus sign.
  • The hex codes of saved colors can be copied to the clipboard by using the clipboard icon.
  • Colors can be removed individually from favorites by selecting the color tile or removed all at once by using the trashcan.
